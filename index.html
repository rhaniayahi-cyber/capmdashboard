Problématique

Les rendements observés des actions correspondent-ils réellement à ceux prédits par le modèle CAPM ? Ce projet vise à analyser la relation entre le risque et le rendement à partir de données financières réelles, puis à comparer les résultats théoriques et observés.

Présentation du projet

Ce projet a été réalisé dans le cadre d’un travail de programmation appliqué à la finance. Son objectif est d’étudier la relation entre le risque et le rendement des actions à l’aide du modèle CAPM, tout en proposant une interface visuelle claire et pédagogique.

Nous avons utilisé Python pour charger les données financières, nettoyer les séries temporelles, calculer les rendements journaliers, estimer les bêtas des entreprises et déterminer les rendements théoriques du CAPM. Ces résultats sont ensuite comparés aux rendements réellement observés.

Contenu du dashboard

Le dashboard est structuré en plusieurs pages complémentaires :

• Simulation : permet de tester différentes stratégies d’investissement en fonction d’un capital initial.
• Résultats : présente les performances, les probabilités de gain et les comparaisons entre stratégies.
• Graphiques : visualise la relation entre le bêta, le rendement et la Security Market Line.

Entreprises étudiées

Les entreprises analysées dans ce projet sont : Apple, Microsoft, Tesla, Airbus, LVMH, TotalEnergies, Air Products, Alibaba, Meta Platforms et Pharmagest.

Le marché est représenté par l’indice SPY (S&P 500), utilisé comme référence pour mesurer le risque systématique.

Intérêt du projet

Le modèle CAPM permet d’évaluer un actif financier en tenant compte à la fois de son rendement et de son niveau de risque. Il offre ainsi un cadre d’analyse permettant de déterminer si une action est sous-évaluée, correctement valorisée ou surévaluée.

Ce projet illustre concrètement cette relation en combinant analyse de données, modélisation financière et visualisation interactive.
